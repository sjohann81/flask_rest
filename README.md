# Simple Flask/SQLAlchemy/Marshmallow REST application

### Environment setup
	$ python3 -m venv env
	$ source env/bin/activate

### Dependencies
	$ pip install Flask Flask-SQLAlchemy Flask-RESTful flask-marshmallow marshmallow-sqlalchemy

### Database creation
	$ python3 create_db.py

### Lauch application
	$ python3 app.py
